# 第8讲 实意动词的否定/提问/回答

## 一、使用助动词进行否定 

> 在助动词 do does did 后面加 not

```
do not / don’t            (一般现在时 )
does not / doesn't     (一般现在时 does是do的第三人称单数现在时形式。)
did not / didn't          (did是do的过去式 )
```

例子:

```
1.I don't  go to school by bus. (我不做公共汽车去学校)

2. She doesn't watch TV everyday. (她每天不看 电视)

3. They didn't swim last night.  (他们 昨晚没有游泳)
```

## 二、使用助动词进行提问

> 助动词放句首，后面的实意动词变原型

```
1. He often  plays golf. (他常常 打高尔夫球。)
Does he often play golf?   (他常常 打高尔夫球吗?)
Yes,he  does. / No,He doesn't.

2.They go to school by bus. (他们搭公交车去学校)
Do they go to school by bus？
Yes,they do. / No,they don't

3. Sam had breakfast yesterday. (山姆 昨天吃早餐了)
Did sam have breakfast  yesterday？ （山姆昨天有吃早餐吗？）
Yes,He did./ No,He didn't


```

## 练习

```
1. 他每天都学英语吗？(learn English)
Does he learn English everyday? 
yes,he does.
He learn English everyday. (他每天都学英语)
2. Tom 昨天没有吃早餐。（have breakfast）
Tom  didn't have breakfast yesterday.
```

## 重点单词

```
do      【 du:】 v.  做，干，办（某事）；行动，表现；顺利进行，进展；从事（工作）；执行（任务）；
does  【dʌz】 v. 做；工作；有用（do 的第三人称单数形式）
did       【did】v. 做（do 的过去式）
swim    【swim】 v. 游泳；横渡；游（某一泳姿）；（鱼、禽等）游过，游动；浸，泡；漂浮；使漂浮，使渡过；（物体）仿佛旋转；眩晕
                              n. 游泳；（河流中）适合钓鱼的静止深水处，深潭；漂浮；眩晕
                              adj. 游泳时穿戴的
breakfast 【'brekfəst】   n. 早餐；早餐的，早餐时用的  
									v. 吃早餐    
have breakfast   吃早餐，吃早饭                  
```
