# 第六讲 反身代词

> 反身代词，代表某人自己

## 单数形式

> myself  (我自己) 
> yourself (你自己)
>
> herself (她自己)
>
> himself (他自己)
>
> itself (它自己)

## 复数形式

> yourselves  (你们自己)
>
> ourselves (我们自己)
>
> themselves (他们自己/ 她们自己/它们自己)

## 反身代词的用法

> 反身代词有2个作用
>
> 1.作为宾语
>
> 2.作为同位语

 ### 什么是同位语

一个名词(或其它形式)对另一个[名词](https://baike.baidu.com/item/名词/502047)或[代词](https://baike.baidu.com/item/代词/87578)进行解释或补充说明，这个名词（或其它形式）就是同位语。同位语与被它限定的词的格式要一致，并常常紧挨在一起。同位语除表示其同位成分的全部意义外，还可以表示部分意义

### 例子

```

1. Please help yourself to some fish.  请随便吃点鱼。

2. We enjoyed  ourselves last night.  昨晚我们玩得很开心。（我们享受我们昨天晚上）

3.The  thing itself is not important. 这件事本身并不重要。
```

## 练习

```
1.Take good care  of （yourself ).    好好照顾 (你自己)。

2. She  gained control of  (herself ). 她控制住了 (她自己)。
```
